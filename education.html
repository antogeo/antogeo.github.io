Title: Machine Learning applications for characterizing brain-damaged patients’ level of consciousness
Brief Synopsis: In my thesis I applied Machine Learning techniques on functional neuroimaging data for characterizing the level of consciousness of patients with damaged brains. The results provided important information for the neural correlates of consciousness. Additionally, a Python library for processing medical images, which can be used as a complementary tool to the behavioral assessments, was developed and validated.
